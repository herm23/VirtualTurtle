FIRST RUN THE SIMULATION AND APRILTAG-ROS (appropriate launch files)
ros2 launch ir_launch assignment_1.launch.py
ros2 launch group28_assignament_1 launch_apriltag.yaml

THEN IN A SEPARATE TERMINAL RUN THE tag_detector_node
ros2 run group28_assignament_1 tag_detector_node

SEND A FAKE REQUEST TO THE tag_detector_node
BEWARE: do not request it from map frame (issues with tf tree in simulation)

ros2 service call /detect_tags group28_assignament_1/srv/DetectTags "{ids: [1,10], target_frame: 'odom'}"


SEND IT AS GOAL VIA COMMAND LINE
BEWARE: ask it in odom or base_link, convert it to map

ros2 topic pub --once /goal_pose geometry_msgs/PoseStamped "{
    header: {
        stamp: {sec: 1211, nanosec: 301000000},
        frame_id: 'map'
    },
    pose: {
        position: {x: 11.730387, y: -2.538328, z: 0.0},
        orientation: {x: 0.0, y: 0.0, z: 0.0, w: 1.0}
    }
}"
